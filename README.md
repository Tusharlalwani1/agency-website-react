## 🛠️ Tech Stack

- ⚛️ React.js
- ⚡ Vite
- 🎨 CSS3
- 🌬️ Tailwind CSS
- 🧩 JavaScript (ES6+)

---

## ✨ Features

- Fully responsive design
- Modern UI/UX layout
- Reusable components
- Hero section with CTA
- Trusted clients section
- Services / work showcase
- Contact form UI
- Dark/Light theme toggle
- Fast performance with Vite

## 🚀 Getting Started

### 1. Clone repo

```bash
git clone https://github.com/Tusharlalwani1/agency-website-react.git
cd agency-website-react
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run dev server

```bash
npm run dev
```

---

## 📦 Build

```bash
npm run build
```
